## 🎯 迁移指南

### 从v1.2升级到v1.3

1. **更新代码**
   ```bash
   git pull origin main
   pip install -r requirements.txt
   ```

2. **配置迁移**
   ```bash
   # 添加新的配置项到.env
   echo "LLM_PROVIDER=deepseek" >> .env
   echo "DEEPSEEK_API_KEY=your_key" >> .env
   ```

3. **测试环境**
   ```bash
   python test_llm.py
   python setup_env.py
   ```

### 从v1.1升级到v1.2

1. **创建配置文件**
   ```bash
   cp env.example .env
   # 编辑.env文件，填入API密钥
   ```

2. **更新代码引用**
   - 将硬编码的配置替换为`config.xxx`
   - 更新文件路径引用
